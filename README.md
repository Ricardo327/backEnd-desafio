## Instalação

Clone the repo locally:

```sh
git clone https://github.com/Ricardo327/backEnd-desafio.git
cd backEnd-desafio
```

Instalar PHP dependencias:

```sh
composer install
```

Copiar e configurar o arquivo .env:

```sh
cp .env.example .env
```

Iniciar o servidor:
```sh
php artisan serve
```
